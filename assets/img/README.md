# Carpeta de imágenes

Esta carpeta queda preparada para alojar las imágenes del proyecto.

Actualmente el `index.html` conserva las rutas remotas de WordPress para que la página mantenga el diseño original sin romperse por falta de archivos locales.

Cuando descargues las imágenes, puedes guardarlas aquí y actualizar las rutas del HTML, por ejemplo:

```html
<img src="assets/img/logo-computintas.png" alt="Computintas Digital">
```
